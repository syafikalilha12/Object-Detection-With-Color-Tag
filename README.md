# Object-Detection-With-Color-Tag
Create a program to detect color tags with the following definitions: The order of the circles is circle 1 from the top left corner, circle 2 from the top right corner, so there are numbering 6 circles. Each circle has the possibility of 3 colors, namely Red, Green, Blue. The combination of the colors in the circle will indicate the meaning of the color tag.
